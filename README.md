This application has used Node.js and Express.js in order to run this app on your system, 
you need to have Node version 20 and above. Follow the steps given below to run the app.

Step 1: Create a folder (if you've downloaded the frontend for this before, then open a new terminal
and follow the steps in the same folder, where you've downloaded the frontend.

Step 2: Open the terminal and type 
        git init 
        The above command will initialize a git repository in your folder

Step 3: Type (or copy-paste this command ) 
        git clone https://github.com/DumplingsSushi/Backend

Step 4: Type (or copy-paste this command ) 
        cd Backend
        This will take you to the code
        
Step 5: Type (or copy-paste this command )
        npm start
        This will run the project and provide a link, clicking on which will take you to the running website 
        (To use the website, make sure to get the frontend running to make API calls and use the UI 
        to interact.)
